JavaScript基础

1 如何判断一个变量的类型

1.1 总结

如果要判断基本数据类型或JavaScript内置对象，使用 toString ；如果要判断自定义类型，使用 instanceof 。

1.2 解释

在ES5中，变量分为基本类型（五种，分别为字符串、数字、布尔值、null和undefined）和引用类型（Object）。而常见的类型检查手段有：typeof、instanceof、constructor和toString。这些手段各有其优缺点：

  • typeof：只能检测基本数据类型，并且对其中的null还有bug。
  • instanceof：用于检查某个对象的原型链是否包含某个构造函数的 prototype 属性。适用于检测自定义的引用类型对象，它是基于原型链运作的。
  • constructor：指向的是最初创建者，而且容易伪造，不适合做类型判断。
  • toString：使用 Object.prototype.toString.call() 来检测，适用于ECMA内置JavaScript类型（包括基本数据类型和内置对象）的类型判断。

2 深复制和浅复制的区别

深复制和浅复制只针对复杂对象，可以说，浅复制只复制一层对象的属性，而深复制则递归复制了所有层级。

3 作用域链和原型链

作用域链是用来解析标识符的，它是词法作用域的一种实现方式。
原型链是用来查找对象属性的，它是原型继承的一种实现方式。
