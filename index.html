Visual Studio Code入门

发表于 2017-01-12 |

由于才开始学习VSC（以前大多用Sublime Text），所以对其用法和特性都不了解。决定学习因为自己的两个感受：一是觉得VSC相比Sublime Text更好看，二是觉得VSC相比于Sublime Text更像个IDE。下面是我学习VSC文档时记录的笔记：

1 功能区

1.1 功能区一：5大主菜单

1.1.1 Explore-文件夹

  • 功能：对文件夹和文件的管理
  • 打开：Ctrl + 0

1.1.2 Search-搜索

  • 打开：设置快捷键win + 2,可直接定位到搜索框

1.1.3 Git-版本管理

  • 推荐：使用命令行，《Github入门与实践》总结一遍。

1.1.1 Debug-调试

  • 方法
    • 打开要调试的文件，按F5，编辑器会测绘生成一个launch.json
    • 修改launch.json的name和program字段，改成和项目对应的
    • 点击编辑器左侧长得像蜘蛛的那个按钮
    • 点击左上角DEBUG后面的按钮，启动调试
    • 打断点，尽情调试（只要你会chrome调试，一模一样）

1.1.1 Plugin-插件

1.2 功能区二：二级导航

点击每个主菜单都会显示对应的二级菜单。

1.3 功能区三：编辑区

  • 代码
  • 文件名称
  • 断点
  • 分屏
  • git历史
  • 关闭
  • 代码位置和警示信息

1.4 功能区四：信息显示区

当前git信息、格式和字符编码等
